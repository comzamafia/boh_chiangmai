📌 Recipe Name

Use a clear, unique name. Include the protein if it's a variation (e.g. "Pad Thai Chicken", "Pad Thai Shrimp"). This name appears on reports and cost analysis.

📌 Category

Select from the dropdown (e.g. Noodles, Main Dishes, Sauces, Beverages). Categories are managed by admin. If your category is missing, ask admin to add it.

📌 Image URL

Paste a link to a photo (from Google Images, Unsplash, etc.). Right-click any image → Copy image address. Leave blank if no photo available.

ℹ️ Main Sauce checkbox

Check this if the recipe is a base sauce used across multiple dishes (e.g. Nam Prik Pao). It gets a special gold badge on the recipe card.

⬡ Sub Recipe checkbox

Check this if the recipe is used as an ingredient in other recipes (e.g. a homemade curry paste). The system will automatically add it to the Ingredients list with the full calculated cost per unit.

📌 Yield Amount + Unit

Amount: How many this recipe makes (usually 1 for single-portion dishes, or the batch size for sauces e.g. 500).
Unit: portion, g, ml, kg, L, piece, etc.

📌 Selling Price (Dine-in)

The menu price customers pay when eating in. This is the main price used to calculate Food Cost %. Required for FC% to show.

📌 Delivery Price

Optional. If your delivery price is different (e.g. GrabFood, LINE MAN), enter it here. The system calculates FC% for both separately.

✅ Example: Pad Thai (1 portion)

  • Yield: 1 portion
  • Selling price: ฿120 (dine-in)
  • Delivery price: ฿135
  • Prep time: 10 min
  • Cook time: 8 min

⚠️ Example: Sauce / Batch Recipe

  • Yield: 500 g (whole batch)
  • Selling price: typically leave blank — set it as a Sub Recipe instead so other dishes can use it

If you can't find an ingredient, it hasn't been added to the system yet. Go to 🥩 Ingredients in the sidebar and add it there first, then come back to this recipe.

📌 Labor Cost / Hour (฿)

The hourly wage of the cook making this dish. Default is ฿50/hr. The system auto-calculates: (prep + cook time) ÷ 60 × hourly rate

📌 Energy Cost / Batch (฿)

Gas + electricity cost to cook one batch. Default is ฿2. Adjust for dishes that use more fuel (deep fry, long simmer, etc.)

🧮
Labor formula: (PrepTime + CookTime) ÷ 60 × LaborCostPerHour
Example: (10 + 8) ÷ 60 × ฿50 = ฿15.00

✅ FC% Color Guide

  • 🟢 Under 30% — Excellent margin
  • 🟡 30–35% — Good, within target
  • 🟠 35–40% — Acceptable but review
  • 🔴 Above 40% — Too high, adjust price or reduce cost

ℹ️ Defaults are system-wide

The default labor rate (฿50/hr) and energy cost (฿2) are set per recipe — you can adjust each recipe independently to reflect actual complexity.

Quick Reference

Complete Checklist

Use this checklist every time you add a new recipe to make sure nothing is missed.

📋 Basic Info

  • ✓ Recipe name is clear and unique
  • ✓ Category is selected from dropdown
  • ✓ Image URL added (if available)
  • ✓ Cooking instructions written (optional)
  • ✓ "Main Sauce" checked if applicable
  • ✓ "Sub Recipe" checked if used in other recipes

⚖️ Yield & Pricing

  • ✓ Yield amount entered (e.g. 1)
  • ✓ Yield unit selected (portion / g / ml)
  • ✓ Selling price (dine-in) entered in ฿
  • ✓ Delivery price entered (if different)
  • ✓ Prep time entered in minutes
  • ✓ Cook time entered in minutes

🥩 Ingredients

  • ✓ All ingredients are in the Ingredients list
  • ✓ Each ingredient added via picker (not typed manually)
  • ✓ Quantities match the recipe unit shown
  • ✓ No ingredients accidentally duplicated
  • ✓ Ingredient cost in right panel looks correct

💰 Cost Parameters

  • ✓ Labor cost/hr set to actual wage rate
  • ✓ Energy cost/batch adjusted for dish type
  • ✓ FC% is below 35% (ideally)
  • ✓ If FC% > 40%, review ingredients or price
  • ✓ Recipe saved (button shows "✓ Saved")

Best Practices

Tips for New Users

🍽️

Name recipes consistently

Use the format "[Dish] [Protein]" for easy comparison. Example: "Pad Thai Chicken", "Pad Thai Shrimp", "Pad Thai Tofu" — all visible together in reports.

⎘

Use Duplicate for Variations

Never type a full recipe twice. Click the ⎘ copy button on any recipe card to duplicate it, then just swap out the protein ingredient and recalculate FC%.

📊

Check Analysis after saving

Go to 📈 Analysis → Recipe Costs to see all recipes ranked by Food Cost %. This shows you which dishes are most and least profitable at a glance.

⬡

Sub Recipe for sauces

If you have a base sauce (e.g. Massaman curry paste), mark it as a Sub Recipe. It becomes available as an ingredient in other recipes with the exact cost included.

⚠️

FC% above 40%?

Options: (1) Reduce portion size, (2) Substitute a cheaper ingredient, (3) Increase selling price, (4) Check if ingredient prices are outdated in the Ingredients page.

🔒

Can't see all recipes?

Your Admin may have restricted your account to specific Recipe Categories. This is normal — you only see what's relevant to your department. Ask Admin to adjust access.

🏆

Target Food Cost Benchmarks

Under 28% — Beverages & Desserts
28–33% — Noodles & Rice dishes
33–38% — Meat & Seafood mains
Above 40% — Review & adjust
